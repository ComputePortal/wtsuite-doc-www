Getting started

A simple example to get you acquainted with wtsuite's typed javascript.

Hello world

Create a file name example.wts with the following content:

var message String = "Hello world";
// message = 0.0; // uncomment this line to see the transpiler throw a type error
console.log(message);

Transpile example.wts:

> wt-script example.wts -o example.js --target nodejs

The resulting javascript will have the following content:

'use strict'
...
console.log('Hello world');

Note that a header is added, comments are removed from the output, and different literal string quotes are used. Single quotes are used by default in the transpiler output for literal strings, but double quotes are used if the literal string contains a single quote itself.


The output can be run using nodejs:

> nodejs example.js
Hello world
