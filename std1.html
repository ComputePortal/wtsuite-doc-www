std/layout

FlexStyle

Example usage:

import * as layout from "std/layout"

div(style=layout.FlexStyle(hlc)) // display: flex, direction: row, left aligned, vertically centered

The argument to layout.FlexStyle consists of three characters:

  1. h (flex-direction: row) or v (flex-direction: column)
  2. l (left aligned), c (horizontally centered), r (right aligned)
  3. t (top aligned), c (vertically centered), r (bottom aligned)

In case of h the second character can be s (justify-content: space-between). Similarly in case of v the third character can also be s.

hor

The layout.hor tag extends the div tag, and accepts a two-letter argument, specifying the horizontal and vertical alignment (exactly like the second and third characters of the argument to layout.FlexStyle.

ver

The layout.ver tag extends the div tag, and accepts a two-letter argument, specifying the horizontal and vertical alignment (exactly like the layout.hor tag.

GridStyle

Example usage:

import * as layout from "std/layout"

div(style=layout.GridStyle(gutter=1rem, nColumns=12)) // display: grid, grid-template-columns: [1fr, 1fr, ...]

contentdiv

Example usage:

import * as layout from "std/layout"

layout.contentdiv(maxWidth=1400px)

Children of layout.contentdiv are centered on the page, and limited in width.

multicol

Example usage:

import * as layout from "std/layout"

layout.multicol(rowHeight=1.2rem, screenWidths=[700px])

The entries in screenWidths specify at which points an additional column is allowed to be displayed.


This only works properly if the height of each child is the same, and known in advance.
