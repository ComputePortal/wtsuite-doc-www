Templates

Templates allow you to reuse groups of tags. You will typically use this for creating templates of pages.

Page example

Any composition of tags can be reused by defining a template:

// file: page.wtt
export template page(title) extends html super()
head
meta(charset=utf-8)
title
$title
body
block default

The template can then be instantiated as follows:

// file: page-instance.wtt
import {page} from "./page.wtt"

page(title="web transpiler suite")
h1 "Welcome"

Any child content of the new page tag is placed under the default block. If no default block is defined then the content is simply appended to the end.

Blocks

To explicitely append to a named block, you can use the append statement:

page(title="web transpiler suite")
append default
h1 "Welcome"

You can also replace the content of a tag using the replace statement:

// file: page.wtt
export template page extends html super()
head
meta(charset=utf-8)
title
block title
"web transpiler suite"

// file: page-instance.wtt
page
replace title
"Web Transpiler Suite!"
