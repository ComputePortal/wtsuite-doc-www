Download

The wtsuite tools can be compiled from source, or the binaries can be installed directly for MSWindow, MacOS or Linux (64bit).


Syntax coloring and indentation is included for vim.

Latest binaries

Pick the package that is appropriate for your system:

  • wtsuite_0.5.0-linux_x86_64.tar.gz
  • wtsuite_0.5.0-ms_windows.zip
  • wtsuite_0.5.0-mac_darwin.tar.gz

Once you've unpacked the binaries you must place them in your path (see details below).

MacOS installation

Should be similar to the linux installation, see below.

Linux installation

Unpack the binaries into a directory included in your $PATH, typically /usr/local/bin


Alternatively use $HOME/bin:


> mkdir -p $HOME/bin
> export PATH=${PATH}:$HOME/bin

MS Windows installation

Unzip the downloaded package into a location that you are able to remember.


Look for the Environment variables menu inside the System properties menu of the System section of the Control Panel.


Add the directory where you unpacked the binaries to the system variable named Path (separated by a semicolon from any previous paths).

Compiling from source

We recommend you use a linux based environment for this.


The source code can be download from our github repository.


Running make in the root directory should do the trick. The resulting files are placed in the build directory.


Running make install compiles the tools and installs the binaries into /usr/local/bin

IDE plugins

Only syntax coloring/indentation for vim is currently provided.

VIM

The vim files can be downloaded here.


The indent files should be copied into the $HOME/.vim/indent/ directory, the syntax coloring files should be copied into the $HOME/.vim/syntax/ directory.


Don't forget to append the following lines to your vimrc file:

au BufNewFile,BufRead *.wtt set filetype=wtt
au BufNewFile,BufRead *.wts set filetype=wts
au BufNewFile,BufRead *.glsl set filetype=glsl

(If you can think of better file extensions, please let us know.)
