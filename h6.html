Looping and branching

The template script language has for loops, if-else directives, and switch directives.

For loop

An example for loop looks like follows:

ul
for item in $items
li $item

If you also need the index (0-based!), you can specify two names, separated by a comma, before the in keyword:

ul
for i, item in $items
li $str($i) + ". " + $item

If else

if and elseif directives take boolean condition as an argument. elseif and else directives must be preceded by an if directive on the same indentation level:

body
if $title != null
h1 $title
else
h1 $defaultTitle

Switch

A switch is good alternative to an if-else directive if there are many similar cases:

switch $tag
case "div"
div
case "form"
form
case "section"
section
default
$error("tag type ", $tag, " not handled")
