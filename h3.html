Operators

An overview of the operators the template script language.


Operators can be used in conventionaly manner, or via their internal function names.


OperatorDescriptionEquivalent function
+Add numbers, join strings, concatenate lists, or merge dicts.add(a, b)
&&Shortcircuit evaluation of two boolean expression.and(a, b)
/Divides two numbers. Both numbers must have the same units, or the denominator must be unitless. The result is not truncated.div(a, b)
==Equality test for primitives. Can also be used to compare any to null.eq(a, b)
>=Input numbers must have same units, or both be unitlessge(a, b)
[]Indexing operatorget(container, key)
>Input numbers must have same units, or both be unitlessgt(a, b)
cond ? a : bIf $cond==true then a is evaluated and returned, otherwise b is evaluated and returned.ifelse(cond, a, [second-cond], b, [...])
<=Input numbers must have same units, or both be unitlessle(a, b)
<Input numbers must have same units, or both be unitlesslt(a, b)
%Input numbers must have same units, or rhs must be unitless.mod(a, b)
*Multiples two numbers. The rhs number must be unitless.mul(a, b)
!=Not equals for primitives. Can also be used to compare any to nullne(a, b)
Unary -Negateneg(a)
!Notnot(a)
-Subtract two numbers. Must have same units. The - must be surrounded by spaces.sub(a, b)
||Shortcircuit boolean oror(a, b)
!!If-not-null-then operator, more informationSyntactic sugar for ifelse(a!=null, b, a)
??Nullish coalescing operator, more informationSyntactic sugor for ifelse(a==null, b, a)
