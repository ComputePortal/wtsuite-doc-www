Getting started

A minimal example to get you started with wtsuite's template script.

Basic page example

Put the following code in a file named example.wtt:

var title = "web transpiler suite"

// !doctype is automatically added
html
head
meta(charset=utf-8)
title
print $title
body
h1 "Welcome"

Transpile this into an html file using the wt-template tool:

> wt-template example.wtt -o example.html

If you omit the -o argument, the result is written to a.html.
